GITHUB PAGES GÜNCELLEMESİ
1. Programdan Ayarlar / Yedek bölümüne girip güncel JSON yedeği alın.
2. GitHub'da 34tufan34/servis-yonetim deposunun ana dizinine girin.
3. + > Upload files seçeneğini açın.
4. index.html, sw.js, KURULUM.txt ve PROJE_HAFIZA_NOTU_v4_32_0.txt dosyalarını yükleyin.
5. Commit directly to the main branch seçili kalsın.
6. Commit changes düğmesine basın ve GitHub Pages dağıtımının tamamlanmasını bekleyin.
7. Tablette uygulamayı tamamen kapatıp yeniden açın.
8. Eski sürüm kalırsa Ayarlar / Yedek > Çevrimdışı Dosyaları Güncelle işlemini kullanın.
