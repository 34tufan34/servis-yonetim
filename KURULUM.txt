SERVİS YÖNETİM SİSTEMİ v4.30.0 - PWA / LİSANSLI SÜRÜM

GITHUB PAGES KURULUMU
1. Bu paketteki dosyaları GitHub deposunun ana dizinine yükleyin.
2. Ana uygulama dosyası index.html'dir.
3. GitHub Pages yayınlandıktan sonra uygulamayı HTTPS adresinden açın.
4. İlk açılışta lisans kodu istenir.

LİSANS SİSTEMİ
- Lisans API: https://servis-lisans-api.ops-429.workers.dev
- Lisans, cihaz bazında Cloudflare D1 veritabanından doğrulanır.
- İlk etkinleştirmede cihaz lisansa kaydedilir.
- Sonraki açılışlarda lisans otomatik kontrol edilir.
- İnternet kesilirse son başarılı doğrulamadan itibaren 7 gün çevrimdışı kullanım sağlanır.
- Araç, personel, tahsilat, finans ve evrak kayıtları lisans sunucusuna gönderilmez.
- Operasyon verileri cihazın kendi localStorage / IndexedDB alanında kalır.

GÜNCELLEME
- index.html, sw.js ve diğer uygulama dosyaları GitHub'da güncellendiğinde bütün kullanıcılar yeni program sürümünü alır.
- Her kullanıcının cihazındaki işletme verileri birbirinden bağımsız kalır.
- GitHub Pages adresi, APK paket adı ve APK imzası değiştirilmemelidir.

YEDEKLEME
- Önemli değişikliklerden önce JSON yedek alın.
- Araç evraklarının Blob dosyaları JSON yedeğe dahil değildir; ayrıca saklanmalıdır.
- Uygulama kaldırılırsa veya cihaz verileri temizlenirse yerel kayıtlar kaybolabilir.

PWA KURULUMU
1. index.html dosyasını Chrome/Edge ile HTTPS üzerinden açın.
2. Ayarlar / Yedek > Uygulamayı Kur bölümüne girin.
3. "Uygulamayı Kur" butonuna basın.

Hızlı yerel test:
- Bu klasörde terminal açın.
- Python varsa: python -m http.server 8080
- Tarayıcıdan: http://localhost:8080
